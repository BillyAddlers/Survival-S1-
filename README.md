# Azalel Nation Survival [S1]

Selamat datang di Repositori GitHub Azalel Nation Survival [S1] !

Di repositori ini anda dapat menemukan change log atau yang biasa kita sebut sebagai catatan perubahan server dari waktu ke waktu yang akan ditulis manual oleh Developer. Change log ini sangat berguna untuk mengetahui perubahan penting pada server dan dapat berguna untuk ke depannya. Untuk melihat catatan perubahan silakan buka file `changelog.log`.

Bagi developer/administrator, apabila ingin menambahkan suatu statement pada change log, silakan gunakan format berikut:
[Tanggal waktu perubahan]
+ [Add . . .]
- [Remove . . .]
